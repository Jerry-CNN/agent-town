<objective>
Configure PixiJS to preserve pixel-art crispness by setting scaleMode to 'nearest' before any texture loads, adding CSS image-rendering: pixelated, and enabling roundPixels on the Application component.

Purpose: Without this configuration, all pixel-art assets loaded in subsequent phases (tilesets in Phase 12, sprites in Phase 13) would render with bilinear filtering, producing blurry edges that destroy the pixel-art aesthetic. This must be set BEFORE any Assets.load() call — setting it after has no effect on already-loaded textures (per STATE.md key decision).

Output: Updated MapCanvas.tsx with TextureStyle initialization, updated index.html with CSS rule, visual confirmation of crisp rendering.
</objective>
