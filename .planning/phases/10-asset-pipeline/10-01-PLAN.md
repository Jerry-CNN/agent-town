**Per D-03:** Create `scripts/convert_sprite_atlas.py` — a Python script using only stdlib (`json`, `pathlib`) that reads the reference Phaser-format sprite atlas and outputs a PixiJS-format spritesheet JSON.

**Source file:** `~/projects/GenerativeAgentsCN/generative_agents/frontend/static/assets/village/agents/sprite.json`

**Source format (Phaser JSONArray):**
```json
{
  "frames": [
    { "filename": "down-walk.000", "frame": { "w": 32, "h": 32, "x": 0, "y": 0 }, "anchor": { "x": 0.5, "y": 0.5 } },
    ...20 entries total...
  ],
  "meta": { "description": "Atlas generado con Atlas Packer Gamma V2", ... }
}
```

**Target format (PixiJS ISpritesheetData):** The output JSON must have this exact structure:

```json
{
  "frames": {
    "down-walk.000": { "frame": { "x": 0, "y": 0, "w": 32, "h": 32 }, "sourceSize": { "w": 32, "h": 32 } },
    "down-walk.001": { "frame": { "x": 32, "y": 0, "w": 32, "h": 32 }, "sourceSize": { "w": 32, "h": 32 } },
    "down-walk.002": { "frame": { "x": 64, "y": 0, "w": 32, "h": 32 }, "sourceSize": { "w": 32, "h": 32 } },
    "down-walk.003": { "frame": { "x": 32, "y": 0, "w": 32, "h": 32 }, "sourceSize": { "w": 32, "h": 32 } },
    "down": { "frame": { "x": 32, "y": 0, "w": 32, "h": 32 }, "sourceSize": { "w": 32, "h": 32 } },
    ... (same pattern for left, right, up — 20 entries total, keyed by filename) ...
  },
  "animations": {
    "down-walk": ["down-walk.000", "down-walk.001", "down-walk.002", "down-walk.003"],
    "left-walk": ["left-walk.000", "left-walk.001", "left-walk.002", "left-walk.003"],
    "right-walk": ["right-walk.000", "right-walk.001", "right-walk.002", "right-walk.003"],
    "up-walk": ["up-walk.000", "up-walk.001", "up-walk.002", "up-walk.003"]
  },
  "meta": {
    "image": "texture.png",
    "format": "RGBA8888",
    "size": { "w": 96, "h": 128 },
    "scale": 1
  }
}
```

**Conversion logic:**
1. Read the source JSON.
2. Convert `frames` from array to dict: key = entry `filename`, value = `{ "frame": { "x", "y", "w", "h" }, "sourceSize": { "w", "h" } }`. Note: the source uses `{ "w", "h", "x", "y" }` order — normalize to `{ "x", "y", "w", "h" }` in output.
3. Build `animations` dict by grouping frame names that match `{direction}-walk.{NNN}` pattern: `down-walk`, `left-walk`, `right-walk`, `up-walk`. Each animation has 4 frames in numeric order (000-003).
4. Set `meta.image` to `"texture.png"` (relative path — PixiJS resolves relative to the JSON URL).
5. Set `meta.format` to `"RGBA8888"`, `meta.size` to `{ "w": 96, "h": 128 }`, `meta.scale` to `1`.
6. Write to `frontend/public/assets/agents/sprite.json` with `indent=2`.

The script must also accept an optional argument for the output path (default: `frontend/public/assets/agents/sprite.json`).

After creating the script, run it: `python3 scripts/convert_sprite_atlas.py`

Then validate the output JSON structure programmatically:
- `frames` is a dict (not a list), with exactly 20 keys
- `animations` has exactly 4 keys: `down-walk`, `left-walk`, `right-walk`, `up-walk`
- Each animation has exactly 4 frame references
- `meta.image` equals `"texture.png"`
- `meta.size` equals `{ "w": 96, "h": 128 }`
  </action>
