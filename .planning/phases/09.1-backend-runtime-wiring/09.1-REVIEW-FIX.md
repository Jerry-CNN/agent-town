**Files modified:** `backend/simulation/engine.py`

**Applied fix:** Moved `timeout = max(self.tick_interval * 2, 120)` to before the `try` block in `_agent_step_safe`, so the `except asyncio.TimeoutError` handler reuses the same captured variable instead of recomputing `self.tick_interval` (which calls `get_adaptive_tick_interval()` and may return a different value).

### WR-02: EVTS-02 — whisper heard_by tracks ALL agents, not just the target

**Files modified:** `backend/simulation/engine.py`
**Commit:** 24603ac
**Applied fix:** Added `ev.target == agent_name` guard in the whisper `heard_by` update loop inside `_agent_step`. Only the intended recipient is now appended to `heard_by`; all other agents running `_agent_step` are skipped by the new condition.

### WR-04: `test_expired_events_removed_after_tick` tests inline logic, not the engine
