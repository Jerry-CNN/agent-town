    Create `backend/simulation/world.py` with these exact classes (per D-05, D-06, D-12, D-14, and research Pattern 1-2):

    1. Module-level constant: `ADDRESS_KEYS = ["world", "sector", "arena"]`

    2. `@dataclass class Tile`:
       - `coord: tuple[int, int]` -- (x, y) grid position
       - `address: list[str]` -- hierarchical path e.g. `["agent-town", "cafe", "seating"]`, default empty list
       - `collision: bool` -- walkability flag, default False
       - `_events: dict` -- runtime event store for Phase 3, default empty dict, repr=False
       - Method `get_address(level: str | None = None, as_list: bool = True) -> list[str] | str`: returns address up to specified ADDRESS_KEYS level (inclusive). If level is None or last key, return full address. Use `ADDRESS_KEYS.index(level) + 1` to slice.
       - Method `get_addresses() -> list[str]`: returns all ancestor address strings starting from depth 2 (e.g. `["agent-town:cafe", "agent-town:cafe:seating"]`). Uses `":".join(self.address[:i]) for i in range(2, len(self.address) + 1)`.
       - Property `is_walkable -> bool`: returns `not self.collision`

    3. `class Maze`:
       - `__init__(self, config: dict)`: reads `config["world"]`, `config["size"]` as [height, width], `config["tile_size"]`. Initializes full `self.tiles: list[list[Tile]]` grid (height rows x width cols), all default walkable. Then iterates `config["tiles"]` sparse list, overwriting tiles with address/collision data. Prepends `self.world` to each tile's address. Builds `self.address_tiles: dict[str, set[tuple[int, int]]]` reverse index from all tile `get_addresses()`.
       - `tile_at(self, coord: tuple[int, int]) -> Tile`: returns `self.tiles[y][x]`
       - `get_walkable_neighbors(self, coord: tuple[int, int]) -> list[tuple[int, int]]`: 4-directional (up/down/left/right). Uses STRICT inequality `0 < c[0] < self.width - 1` and `0 < c[1] < self.height - 1` per reference boundary guard (border tiles are collision by convention).
       - `find_path(self, src: tuple[int, int], dst: tuple[int, int]) -> list[tuple[int, int]]`: BFS using `collections.deque`. Returns `[src]` if src==dst. Uses distance map `dist[y][x]` initialized to 0, marks src as 1. Explores via `get_walkable_neighbors`. If dst unreachable (`dist[dst_y][dst_x] == 0`), returns `[]`. Reconstructs path backward from dst to src using decrementing distance values. Returns reversed path.
       - `get_address_tiles(self, address: list[str]) -> set[tuple[int, int]]`: joins address with ":" and looks up `self.address_tiles`.
       - `resolve_destination(self, sector: str) -> tuple[int, int] | None`: builds key `f"{self.world}:{sector}"`, gets tile coords from `address_tiles`, filters to walkable only, returns `random.choice(walkable)` or None if empty.
