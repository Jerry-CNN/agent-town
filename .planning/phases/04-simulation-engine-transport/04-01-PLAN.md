Create `backend/simulation/engine.py` with:

1. **AgentState dataclass** (per D-09, D-10 research pattern):
   ```python
   @dataclass
   class AgentState:
       name: str
       config: AgentConfig
       coord: tuple[int, int]         # mutable, updated each tick
       path: list[tuple[int, int]]    # remaining BFS tiles, pop(0) each tick
       current_activity: str          # broadcast to clients
       schedule: list[ScheduleEntry]  # modified by conversations
   ```

2. **TICK_INTERVAL constant = 5** (Claude's discretion within D-01 range of 5-10s; 5s for responsiveness)

3. **SimulationEngine class** with:
   - `__init__(self, maze: Maze, agents: list[AgentConfig], simulation_id: str)`:
     - Store maze, simulation_id
     - Store agents list as `self._configs`
     - Create `self._running = asyncio.Event()` (start cleared = paused, per D-07)
     - Initialize `self._agent_states: dict[str, AgentState] = {}`
     - `self._broadcast_callback: Callable | None = None` (set by Plan 02 when wiring ConnectionManager)
     - `self._tick_count: int = 0`

   - `async def initialize(self)`:
     - Call `reset_simulation(self.simulation_id)` to clear stale ChromaDB data (INF-01)
     - For each agent config: create AgentState with config.coord, config.currently, empty path
     - Generate daily schedules for all agents in parallel using `asyncio.TaskGroup` (D-03: 2 LLM calls per agent via generate_daily_schedule)
     - Store schedule results in each AgentState.schedule
     - Store observation memory for each agent's initial state via add_memory

   - `async def run(self)`:
     - Set `self._running` (enter running state)
     - Call `self._tick_loop()`

   - `async def _tick_loop(self)`:
     - `while True:` loop
     - `await self._running.wait()` at top (D-07: blocks when paused)
     - Run all agent steps in parallel with `async with asyncio.TaskGroup() as tg:` + `tg.create_task(self._agent_step_safe(name, state))`
     - Catch `ExceptionGroup` with `except* Exception` and log but NEVER re-raise (loop must survive)
     - `self._tick_count += 1`
     - `await asyncio.sleep(TICK_INTERVAL)`

   - `async def _agent_step_safe(self, agent_name: str, state: AgentState)`:
     - Wrap `_agent_step()` in `try/except Exception` -- log warning on failure, NEVER propagate (critical: prevents TaskGroup from cancelling sibling agents)
     - Use `asyncio.wait_for(self._agent_step(agent_name, state), timeout=TICK_INTERVAL * 2)` to skip agents whose LLM calls exceed 2x tick interval (Claude's discretion for slow LLM handling)

   - `async def _agent_step(self, agent_name: str, state: AgentState)`:
     - Build `all_agents_view` dict: `{name: {"coord": s.coord, "current_activity": s.current_activity} for name, s in self._agent_states.items()}`
     - Call `perceive(agent_coord=state.coord, agent_name=agent_name, maze=self.maze, all_agents=all_agents_view)`
     - **MOVEMENT PHASE** (D-09, D-10): If `state.path` is non-empty, pop first tile with `state.path.pop(0)`, update `state.coord`, call `self._emit_agent_update(agent_name, state)`, and return early (movement tick, no decide call)
     - **CONVERSATION PHASE**: Check `perception.nearby_agents` -- for each nearby agent, call `attempt_conversation()`. If True, call `run_conversation()` and update both agents' schedules from the result. Call `self._emit_conversation(conversation_result)`. Then return (conversation tick, no decide call). Only attempt one conversation per tick per agent.
     - **DECIDE PHASE**: Call `decide_action()` with simulation_id, agent_name, config.scratch, config.spatial, current_activity, perception, schedule
     - If `action.destination != "idle"`: call `self.maze.resolve_destination(action.destination)`, then `self.maze.find_path(state.coord, dest_coord)`. Store path in `state.path` (skip first element which is current position, so `state.path = path[1:] if len(path) > 1 else []`)
     - Update `state.current_activity = action.activity`
     - Call `self._emit_agent_update(agent_name, state)`
     - Store observation memory via `add_memory(simulation_id, agent_name, f"{agent_name} is {action.activity} at {perception.location}", "action", importance=3)`

   - `async def _emit_agent_update(self, agent_name: str, state: AgentState)`:
     - If `self._broadcast_callback` is set, call it with a dict: `{"type": "agent_update", "name": agent_name, "coord": list(state.coord), "activity": state.current_activity}`
     - This is a hook for Plan 02 to attach the ConnectionManager broadcast

   - `async def _emit_conversation(self, conversation_result: dict)`:
     - If `self._broadcast_callback` is set, call it with: `{"type": "conversation", "turns": conversation_result["turns"], "summary": conversation_result["summary"]}`

   - `def pause(self)`: `self._running.clear()` (D-07)

   - `def resume(self)`: `self._running.set()` (D-07)

   - `def get_snapshot(self) -> dict`:
     - Return `{"agents": [{"name": n, "coord": list(s.coord), "activity": s.current_activity} for n, s in self._agent_states.items()], "simulation_status": "running" if self._running.is_set() else "paused", "tick_count": self._tick_count}`
