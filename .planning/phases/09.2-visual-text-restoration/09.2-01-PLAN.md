   ```typescript
   const MAX_ACTIVITY_LEN = 18;
   function truncateActivity(text: string): string {
     if (text.length <= MAX_ACTIVITY_LEN) return text;
     return text.slice(0, MAX_ACTIVITY_LEN - 1) + "\u2026";
   }
   ```
   (The `\u2026` is the ellipsis character.)

3. **Add activityTextRef** inside AgentSpriteInner, alongside existing containerRef:
   ```typescript
   const activityTextRef = useRef<PixiText | null>(null);
   ```

4. **Read initial activity** alongside existing `initialAgent` reads:
   ```typescript
   const initialActivity = truncateActivity(initialAgent?.activity ?? "");
   ```

5. **Add activity text update in useTick** — inside the useTick callback, AFTER the
   lerp position code (after `containerRef.current.y = cur.y;`), add:
   ```typescript
   if (activityTextRef.current) {
     const newActivity = truncateActivity(agent.activity ?? "");
     if (activityTextRef.current.text !== newActivity) {
       activityTextRef.current.text = newActivity;
     }
   }
   ```
   The text-equality guard prevents unnecessary PixiJS text layout on frames where
   activity has not changed.

6. **Add activity pill draw callback** inside AgentSpriteInner, after the drawCircle callback:
   ```typescript
   const drawActivityPill = useCallback((g: PixiGraphics) => {
     g.clear();
     g.setFillStyle({ color: 0x111111, alpha: 0.7 });
     g.roundRect(-60, -30, 120, 16, 4);
     g.fill();
   }, []);
   ```
   Empty deps array because the pill geometry is constant. The semi-transparent dark
   background (0x111111 at 70% opacity) ensures white activity text is readable on all
   sector backgrounds (pre-verified in RESEARCH.md: all composites pass WCAG AA at 8.49-12.30 ratios).

7. **Add pill + text JSX nodes** in the return JSX, ABOVE the circle (before the
   `{/* Colored circle background */}` comment). Insert:
   ```tsx
   {/* Activity text with pill background — above the circle */}
   <pixiGraphics draw={drawActivityPill} />
   <pixiText
     ref={activityTextRef}
     text={initialActivity}
     x={0}
     y={-24}
     anchor={{ x: 0.5, y: 1 }}
     style={{
       fontFamily: "system-ui, sans-serif",
       fontSize: 11,
       fill: 0xffffff,
     }}
   />
   ```
   The `y={-24}` positions the activity text above the 18px-radius circle. The anchor
   `{ x: 0.5, y: 1 }` centers horizontally and aligns the bottom edge of the text to
   that y coordinate.

Do NOT restore a name pill. The research confirms dark navy (0x1a1a2e) name text passes
