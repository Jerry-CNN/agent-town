Semaphore must NOT be module-level — lazy init pattern:
```python
_semaphore: asyncio.Semaphore | None = None
def get_semaphore() -> asyncio.Semaphore:
    global _semaphore
    if _semaphore is None:
        _semaphore = asyncio.Semaphore(3)
    return _semaphore
```

FastAPI lifespan (NOT @app.on_event which is deprecated):
```python
from contextlib import asynccontextmanager
@asynccontextmanager
async def lifespan(app: FastAPI):
    status = await check_ollama_availability()
    app.state.ollama_status = status
    if not status["available"]:
        logging.warning("Ollama not reachable at startup...")
    yield

app = FastAPI(lifespan=lifespan)
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Frontend → FastAPI REST | HTTP requests from browser (localhost dev, no auth in Phase 1) |
| FastAPI → Ollama | HTTP to localhost:11434 — trusted loopback, but response must be validated |
| WebSocket client → server | JSON messages from browser — not yet trusted input |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Spoofing | WebSocket /ws/{session_id} | accept | Phase 1 is local-only, no auth. Session ID is a client-supplied string. Phase 4 will add validation when simulation commands arrive. |
| T-01-02 | Tampering | POST /api/llm/test prompt param | mitigate | Validate prompt length: reject if > 2000 chars. Never interpolate raw prompt into system instructions. |
| T-01-03 | Information Disclosure | Ollama status endpoint returns model list | accept | Model names are not sensitive; this is local-only. No credentials in response. |
| T-01-04 | Denial of Service | Concurrent LLM calls overwhelming Ollama | mitigate | asyncio.Semaphore(3) caps concurrent Ollama calls. Gateway returns fallback if semaphore times out. |
| T-01-05 | Elevation of Privilege | LLM response injected into Python eval | mitigate | NEVER eval() or exec() LLM output. Always parse via Pydantic.model_validate() — structural validation only. |
| T-01-06 | Information Disclosure | app.state.ollama_status stored in process | accept | Low sensitivity: boolean + model name list. No API keys in Phase 1 (Ollama is keyless). |
</threat_model>
